📈 LLM News Alpha

주요 금융 뉴스를 기반으로 종목별 단기 모멘텀 시그널을 추정하는 실험 프로젝트

LLM + Finance Reuters RSS Alpha Prototype

핵심 기능

  • Reuters RSS 기반 뉴스 자동 수집
  • 티커 매핑 + 감성/임팩트 점수화
  • 종목별 알파 점수 집계 및 LONG/SHORT 후보 생성
  • CSV 결과 저장

실행 방법

pip install -r requirements.txt

python src/news_alpha.py --use-llm false

LLM 모드: --use-llm true (OPENAI_API_KEY 필요)
